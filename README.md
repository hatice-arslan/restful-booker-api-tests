# API Testing with Postman
This repository showcases a basic API testing project built with Postman using the Restful Booker API.

## Overview
The project demonstrates foundational API testing workflows such as sending HTTP requests, validating responses, and organizing API operations into a reusable Postman collection.

## API Operations Covered
- Retrieve all bookings
- Retrieve a specific booking by ID
- Create a new booking
- Explore additional booking-related request scenarios

## Postman Requests Included
- GetBookingslds
- GetBooking
- CreateBooking
- UpdateBooking

## Key Skills Demonstrated
- API request creation and execution in Postman
- RESTful endpoint testing
- HTTP status code validation
- JSON response inspection
- Basic collection organization and test workflow practice

## Tech Stack
- Postman
- REST API
- JSON
- Restful Booker API

## Getting Started
1. Download or clone this repository.
2. Import the Postman collection into Postman.
3. Configure any required values if necessary.
4. Run requests individually or execute the full collection.
5. Review the API responses and test outputs.
